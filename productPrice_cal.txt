※ 원가의 흐름에 관한 공식

기본원가와 가공비

- 직접비(기본원가) = 직접재료비 +직접노무비

- 가공비 = 직접노무비 +제조간접비

- 재료소비액(원재료비 or 직접재료비) = 월초재고액 + 당월재료매입액 - 월말재료재고액

- 노무비소비액(직접노무비) = 당월지급액 +당월미지급액 - 전월미지급액

- 제조간접비 = 간접재료비 +간접노무비 + 간접제조경비

- 제품단위당 원가 = 당월제품제조원가 / 완성품수량

- 당기총제조비용 = 직접재료비 + 직접노무비 + 제조간접비(판매비와 관리비 포함X)

- 매출원가 = 기초제품 +당기제품제조원가 -기말제품


- 초재공품+ 당기총재조원가+기말재공품재고액
 

- 제품 제조원가 = 직접재료비+직접노무비+제조간접비+기초제공품재고액-기말재공품 재고액 

                           (기초재공품 + 당기총제조원가 - 기말재공품 )

- 당기총 제조원가 = 원재료비 + 노무비 + 제조경비

- 직접재료비 = 기초재료액 + 당기매앱액 - 기말재료액

- 제조직접비 = 직접재료비 + 직접노무비 + 직접제조경비

- 제조간접비 = 간접재료비 + 간접노무비 + 제조경비

- 제조원가 = 제조직접비 + 제조간접비

- 주원가 = 직접재료비 + 직접노무비

- 총원가 = 제조원가 + 판매비와 관리

- 기말재공품원가 = 당기총제조원가 - 당기제품제조원가

- 당기재료비 = 기초원재료재고액 + 당기원재료매입액 - 기말원재료재고액 (? 확인요망)

- 매출원가 = 기초제품재고액 + 당기제품제조원가 - 기말제품재고액 (기초제품 + 당기 매입 - 기말 제품)

- 매출총이익 = 순매출액 - 매출원가

- 당기총제조비용(당기제조원가) = 직접재료비 + 직접노무비 + 제조간접비

- 당기제품제조원가 = 기초재공품재공액 +당기총제조비용(당기제조원가) - 기말재공품제공액
